This class inherits from the Qt class QThread which allows it to be a parallel thread to the \doxylink{main_8cpp_a0ddf1224851353fc92bfbff6f499fa97}{main()} thread, and also from vtk\+Command which allows it to act as a "{}callback"{} for the vtk\+Render\+Window\+Interactor. This callback functionallity means that once the render\+Window\+Interactor takes control of this thread to enable VR, it can callback to a function in the class to check to see if the user has requested any changes.

\doxysubsection{Detailed Description}
This class inherits from the Qt class QThread which allows it to be a parallel thread to the \doxylink{main_8cpp_a0ddf1224851353fc92bfbff6f499fa97}{main()} thread, and also from vtk\+Command which allows it to act as a "{}callback"{} for the vtk\+Render\+Window\+Interactor. This callback functionallity means that once the render\+Window\+Interactor takes control of this thread to enable VR, it can callback to a function in the class to check to see if the user has requested any changes.
