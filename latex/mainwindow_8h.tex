\doxysubsection{Detailed Description}
Declaration of the \doxylink{class_main_window}{Main\+Window} class for the main application window. 

This header file declares the \doxylink{class_main_window}{Main\+Window} class, which is the central widget of the application. It manages the user interface, the visualization of 3D models using the Visualization Toolkit (VTK), and the interaction with a separate thread responsible for Virtual Reality (VR) rendering.
